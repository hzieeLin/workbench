# [S1] Problem

当前看板功能相对基础，只支持简单的卡片创建、编辑、移动和删除。用户需要更丰富的卡片内容管理和更灵活的视图交互功能。

## 用户需求

1. **卡片内容丰富化**：需要标签系统和评论系统
2. **视图与交互增强**：需要搜索筛选、不同视图模式和卡片排序功能

## 当前功能限制

- 标签实体已存在但未在UI中实现
- 无评论功能
- 无搜索筛选功能
- 只有看板视图，无其他视图模式
- 卡片排序仅支持拖拽，无自动排序

# [S2] Solution overview

实现完整的看板功能增强包，包含：

1. **标签系统**：创建、分配、筛选标签
2. **评论系统**：为卡片添加评论和讨论
3. **搜索与筛选**：按多种条件筛选卡片
4. **不同视图模式**：列表视图、日历视图、时间线视图
5. **卡片排序**：按多种属性自动排序

## 技术约束

- 保持现有Vue 3 + TypeScript + Pinia技术栈
- 使用现有的TypeORM实体和API模式
- 保持现有UI设计风格（深色主题、圆角、阴影等）
- 确保响应式设计，在不同屏幕尺寸下正常工作

# [S3] Label system

## 功能描述

实现完整的标签管理系统，包括：

### 标签管理
- 在看板设置中创建和管理标签
- 每个标签包含名称和颜色
- 支持编辑和删除标签
- 标签颜色选择器提供预设颜色和自定义颜色

### 卡片标签分配
- 在卡片详情模态框中分配标签
- 一张卡片可分配多个标签
- 在卡片预览中显示标签（最多显示3个，多余显示"+N"）
- 支持快速分配/取消分配标签

### 标签筛选
- 在看板顶部添加标签筛选器
- 支持按单个或多个标签筛选
- 筛选结果实时更新
- 显示当前筛选的标签数量

## 数据模型

现有数据模型已支持：
- `Label` 实体：id, name, color
- `CardLabel` 实体：card_id, label_id

## UI组件

1. **LabelManager**：标签管理组件（在看板设置中）
   - 标签列表显示
   - 创建新标签表单
   - 编辑/删除标签操作
   - 颜色选择器

2. **LabelPicker**：标签选择器（在卡片详情中）
   - 标签搜索框
   - 可选标签列表（带复选框）
   - 已选标签显示
   - 创建新标签快捷入口

3. **LabelFilter**：标签筛选器（在看板顶部）
   - 标签下拉选择
   - 多选支持
   - 清除筛选按钮

4. **LabelBadge**：标签徽章（在卡片预览中）
   - 显示标签名称和颜色
   - 最多显示3个标签
   - 溢出显示"+N"

# [S4] Comment system

## 功能描述

为卡片添加评论和讨论功能：

### 评论功能
- 在卡片详情模态框中添加评论
- 支持文本评论（富文本编辑器可选，v1先支持纯文本）
- 显示评论时间和作者（使用默认用户名"用户"）
- 支持删除自己的评论

### 评论显示
- 按时间顺序显示评论（最新的在底部）
- 显示评论者信息（头像+名称）
- 显示评论时间（相对时间：刚刚、X分钟前、X小时前、昨天、具体日期）
- 评论内容支持换行显示

## 数据模型

需要新增Comment实体：
```typescript
@Entity('comments')
export class Comment {
  @PrimaryGeneratedColumn()
  id!: number

  @Column()
  card_id!: number

  @Column()
  author!: string

  @Column({ type: 'text' })
  content!: string

  @CreateDateColumn()
  created_at!: Date

  @ManyToOne('Card', (card: any) => card.comments)
  @JoinColumn({ name: 'card_id' })
  card!: any
}
```

## UI组件

1. **CommentSection**：评论区域组件（在卡片详情模态框中）
   - 评论列表显示
   - 评论表单
   - 评论数量统计

2. **CommentItem**：单条评论组件
   - 作者头像和名称
   - 评论内容
   - 相对时间显示
   - 删除按钮（仅自己可见）

3. **CommentForm**：评论表单组件
   - 文本输入框
   - 发送按钮
   - 字数限制（可选）

# [S5] Search and filter

## 功能描述

实现强大的搜索和筛选功能：

### 搜索功能
- 在看板顶部添加搜索框
- 按标题和描述搜索卡片
- 实时搜索结果（防抖300ms）
- 搜索高亮显示匹配文本

### 筛选功能
- 按标签筛选（多选）
- 按优先级筛选（低、中、高）
- 按截止日期筛选（今天、本周、已过期、未设置）
- 组合筛选条件（AND逻辑）
- 筛选结果数量显示

## UI组件

1. **SearchBar**：搜索框组件
   - 搜索图标
   - 输入框（带清除按钮）
   - 搜索结果数量

2. **FilterPanel**：筛选面板组件
   - 筛选条件分组
   - 每个筛选条件可展开/折叠
   - 清除所有筛选按钮

3. **ActiveFilters**：当前筛选条件显示
   - 已激活的筛选条件标签
   - 每个标签可单独移除
   - 清除所有筛选按钮

# [S6] View modes

## 功能描述

提供多种视图模式：

### 看板视图（现有）
- 卡片按列表分组
- 支持拖拽排序
- 保持现有功能

### 列表视图
- 卡片以表格形式显示
- 显示列：标题、标签、优先级、截止日期、所属列表
- 支持行级操作（编辑、删除）
- 支持列排序
- 分页显示（每页20条）

### 日历视图
- 月视图：按截止日期在日历上显示卡片
- 周视图：按日期显示卡片
- 支持拖拽更改日期
- 点击日期显示当天卡片列表
- 无截止日期的卡片不显示

### 时间线视图
- 按时间顺序显示卡片（从早到晚）
- 显示时间跨度（创建时间到截止日期）
- 支持缩放（日、周、月）
- 适合项目规划和进度跟踪

## UI组件

1. **ViewSwitcher**：视图切换器组件
   - 视图模式图标按钮
   - 当前视图高亮
   - 工具提示

2. **ListView**：列表视图组件
   - 表格布局
   - 列头可排序
   - 行选择
   - 分页控件

3. **CalendarView**：日历视图组件
   - 月历网格
   - 周视图
   - 日期单元格显示卡片摘要
   - 拖拽支持

4. **TimelineView**：时间线视图组件
   - 时间轴
   - 卡片条显示
   - 缩放控制
   - 今日标记

# [S7] Card sorting

## 功能描述

实现多种卡片排序方式：

### 排序选项
- 按创建时间排序（默认）
- 按更新时间排序
- 按优先级排序（高→低或低→高）
- 按截止日期排序（近→远或远→近）
- 按标题字母顺序排序（A→Z或Z→A）

### 排序控制
- 在看板顶部添加排序选择器
- 支持升序/降序切换
- 记住用户的排序偏好（存储在localStorage）
- 排序影响所有视图模式

## UI组件

1. **SortSelector**：排序选择器组件
   - 排序字段下拉选择
   - 升序/降序切换按钮
   - 当前排序状态显示
